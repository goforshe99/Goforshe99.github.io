我的第一篇博客

你好，欢迎来到我的个人博客！这是我写的第一篇文章。

为什么建这个博客？

一直想有一个属于自己的空间，用来记录学习和工作中的点点滴滴。无论是技术上的踩坑经验，还是生活中的感悟，都值得被记录下来。

互联网上有很多现成的博客平台，但我觉得自己动手搭一个更有意义。从零开始，每一个细节都是自己的选择。

搭建过程

这个博客使用纯 HTML + CSS + JavaScript 搭建，没有使用任何前端框架。原因很简单：

  • 轻量快速，不需要加载额外的库
  • 完全可控，每个像素都是自己决定的
  • SEO 友好，搜索引擎可以直接抓取

如果你也想搭建自己的博客，可以试试看这种方式。

好的设计是简单的设计。
